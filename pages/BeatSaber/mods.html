Beat Saber Mods

Here are my many and old BS mods!

Adblock

Adblock is a mod that blocks very distracting, soundpack promotion button in the main menu of Beat Saber. It was an easy mod to make just using a single file with mostly boilerplate code with a single hook I made.


Language: C++

AniPic

AniPic is a modification of an older version of the mod Nya-Quest (repo, newest version). AniPic brought you pictures from an API called some-random-api.ml and displayed them in the main menu. It was a fun mod to make, but I decided to stop working on it as it was a pain to update.


Language: C++

FeetSaber

FeetSaber is a mod that allows you to play Beat Saber with your feet. It was a fun mod to update but as I am half lazy still a newbie (and modding is a bitch) I have decided to stop supporting it for now.


Language: C++
